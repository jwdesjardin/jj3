Next.js Markdown Blog

Description

This is one of the first projects i made in my coding career. It started out at a great way to take notes. I found that i really like the markdown syntax and began taking my notes with it. This turned into a cool project. Fisrt we use next.js getStaticPaths to read the file system and read markdown files. Then we make use of multer to read the content from markdown files. use getStaticProps to pass this content into our SSG pages. Implement javascript search functionality through react useState hooks. And finally style the content with css modules and scss.
